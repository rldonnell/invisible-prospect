Instantly Setup Checklist

  1. Create 6 campaigns in Instantly (one per bucket above)
  2. For each campaign, add the 3-email sequence with the specified day delays
  3. Add all custom variables to each campaign's settings
  4. Configure sending schedule, warm-up, and daily limits in Instantly
  5. Register each campaign in VisitorID via POST /api/admin/campaigns with the Instantly campaign IDs
  6. Set INSTANTLY_API_KEY and INSTANTLY_PUSH_ENABLED=true in Vercel env vars
  7. Test with a dry run: curl -H "Authorization: Bearer $CRON_SECRET" "https://your-app/api/cron/push-instantly?dry=true"
